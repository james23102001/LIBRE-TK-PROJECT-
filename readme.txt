***Roles:
Ceci est une application de gestion de caisse. Il est possible de mettre de l'argent dans la caisse, 
d'effectuer une dépense, de gérer les utilisateurs et d'imprimer des états de dépenses.

***Requirements:
Il faut installer au préalable un serveur local avec une version de MYSQL récente.

***Message:
Il vous faut noter, pour l'utilisation de ce programme :
	- Demandez à votre technicien de vous installer le logiciel.
	- L'utilisateur par défaut est user avec le mot de passe 1234. C'est un directeur.
	- Les directeurs peuvent tout faire sur le système, sauf imprimer l'historique.
	- Les caissiers peuvent effectuer des dépenses et imprimer la liste des dépenses.
	- Vos techniciens pourront vous tirer l'historique en cas de problème.
	Ce programme est gratuitement distribué à tous. Toutefois, vous pouvez nous
contacter, afin qu'on le personalise pour votre entreprise !
